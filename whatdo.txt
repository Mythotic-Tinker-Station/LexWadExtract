


Prerequisite: Add the doom2.wad to the \tools\extractor\ folder

-------------------------------------------
-- Repeat these steps for each mapset
-------------------------------------------
1:  Copy the wad you want to convert to root

2:  edit the run.bat to and add the wad name you want to convert and the 4 letter ALL CAPS acronym for the mapset
    should look something like this: start Love2D/love.exe "src" "mayhem17.wad" "MAYH"

4:	Click Run.Bat

5:	Watch it do the thing.

6:	Once its done, the \pk3\ folder should have all the mapset assets set and ready along with the rest

7:  Just copy all that stuff to your pack pk3 folder, then setup the MAPINFO, LAUNAGE, and ACS Script
